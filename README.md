# Spot-Find

Iac(Pulumi) & MicroService & ServerLess で開発してみる<br>
Cloud は一旦 GCP で開始

# Requirement

- Pulumi

```
brew install pulumi/tap/pulumi
```

- gcloud

```
brew install --cask google-cloud-sdk
```

- nvm

```
brew install nvm
```

# Installation

```bash
make setup
```
